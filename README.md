GALAXY
======

Random Galaxy Generator: Pops up a window with a random galaxy!

![A Randomly Generated Galaxy](./ss.png)

Requirements
-------------

* SDL 2.0
* SDL Image
* LibPNG
* Gnu Make
* Clang

Configuration
-------------

No config for now, you can just root around in `galaxy.c`.


Usage
---------

    make run

or

    make
    ./galaxy
